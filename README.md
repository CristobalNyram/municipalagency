
# Muncipal Agency

This project aims to improve the agency's revenue and results management.


## Required
- PHP Version 7.3>
- Laravel 8
- MySQL
- tailwindcss

## Installation

Install my-project with npm

```bash
  npm install my-project
  cd my-project
```
    
## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
